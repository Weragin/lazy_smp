## Running server
### Installing dependencies

`pip install -r requirements.txt`

`python main.py`
## Running with pypy
### Installing dependencies

`pypy3 -m pip install -r requirements_pypy.txt`
### Starting server

`pypy3 main_pypy.py`


## Running Tests

### Unit Tests

`python -m unittest test.py`
### Bratko-Kopec Test

`python test_bratko_kopec.py`
